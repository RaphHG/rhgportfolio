Projet réaliser : Déployment automatiser de poste informatiques via WDS et MDT

Durant le stage de première années j'ai eu l'occasion de travailler sur un projet qui est de déployer des postes informatiques via le rôle WDS (Windows Deployment Services) + la solution MDT (Microsoft Deployment toolkit). le but étant que à la fin du projet on puisse déployer des machines sans intervention humaine,
tous cela est rendu possible avec les options de personalisation de MDT.

J'ai du configurer un serveur HPE Proliant DL360 pour installer VMWare ESXi(Hyperviseur de type-1 ou natif, permet à plusieurs SE de fonctionner en parallèle), et d'y installer une MV de Windows Server 2019, sur ce Windows Server j'ai configurer les rôles d'AD, DHCP, WDS et MDT.

  • Configuration du DHCP afin d'envoyer les options 66 et 67 (option PXE).
  • création d'une séquence de tâche avec MDT.
