Déployment de poste

Mise en place d’un serveur Windows avec les rôles Active Directory (AD), DHCP, WDS (Windows Deployment Services) et MDT (Microsoft Deployment Toolkit).
WDS et MDT permettent de déployer automatiquement des postes informatiques vierges via le protocole PXE (Pre-boot eXecution Environment), c’est-à-dire un démarrage réseau (boot via Ethernet). Avec MDT, j'ai crée une séquence de tache, qui permet d'automatiser l'installation du poste sans intervention humaine. à la fin de l'installation le poste est totalement configurer connecter au domaine du projet.

Monitoring de 2 Windows Server
